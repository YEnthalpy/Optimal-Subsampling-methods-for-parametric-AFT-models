# osmac-parametric-aft-models
Code for the paper: Optimal subsampling for parametric accelerated failure time models with massive survival data.
## `function.R`
This R file includes functions to calculate estimations, subsampling probabilities and standard errors.
## `summary.R`
This R file includes functions to generate simulation dataset and summary results.
## `simulation.R`
Codes to reproduce simulation results in the paper.
## `real.R`
Codes to reproduce the results of real data analysis in the paper.
